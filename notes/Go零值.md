# Go零值

Go语言中的零值是变量没有做初始化时系统默认设置的值。

各类型对应的零值如下表所示：

| 数据类型  | 零值  | 说明                        |
| --------- | ----- | --------------------------- |
| bool      | false |                             |
| string    | ""    | 空字符串                    |
| map       | nil   | 键值对类型的变量的零值是nil |
| int/float | 0     | 所有的数字类型的零值都是0   |
